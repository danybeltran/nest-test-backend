## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

Add the `DATABASE_URL` environment variable to a `.env` file

Then

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
